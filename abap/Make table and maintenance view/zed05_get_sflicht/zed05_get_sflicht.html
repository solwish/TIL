FUNCTION zed05_get_sflicht.
*"----------------------------------------------------------------------
*"*"Local Interface:
*"  IMPORTING
*"     REFERENCE(I_CARRID) TYPE  SFLIGHT-CARRID
*"  EXPORTING
*"     REFERENCE(E_CODE) TYPE  CHAR1
*"     REFERENCE(E_MESSAGE) TYPE  CHAR100
*"  TABLES
*"      T_LIST STRUCTURE  SFLIGHT
*"----------------------------------------------------------------------

*       Global data declarations
  CLEAR : t_list, t_list[].
*  테이블즈에 있는 놈은 자동으로 헤더 가지고 있음.

*  Check Input Parameter
  IF i_carrid IS INITIAL.
    e_code    = 'E'.
    e_message = 'Input Carrid'.
    EXIT.
  ENDIF.

  SELECT carrid connid fldate price currency planetype
  INTO CORRESPONDING FIELDS OF TABLE t_list
  FROM sflight
  WHERE carrid = i_carrid.

  IF sy-subrc NE 0.
    e_code      = 'E'.
    e_message   = 'No data found'.
    EXIT.
*    루프 안에서 exit 있으면 루프문 벗어남. 지금은 펑션을 나감. 즉 한단계 나가는 거야.
*    리브리스트 프로세싱은 프로세싱을 스탑하는거야.
  ENDIF.




ENDFUNCTION.
